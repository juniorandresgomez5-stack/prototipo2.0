Prototipo educativo de realidad aumentada web

Esta prueba de concepto demuestra que es posible visualizar contenido 3D educativo desde el navegador, usando la camara del dispositivo y un marcador visual, sin instalar aplicaciones adicionales.

Acceso simple Abre el enlace desde tu navegador movil o de escritorio.
Uso de camara Permite el acceso a la camara cuando el navegador lo solicite.
Marcador Hiro Apunta la camara al marcador para activar la experiencia RA.
Ir a la experiencia WebAR

Como probar el prototipo

  1. Abre esta pagina y concede permiso para usar la camara.
  2. Desplazate hasta la experiencia WebAR.
  3. Apunta la camara al marcador Hiro.
  4. Observa el modelo 3D y muevete alrededor para verlo desde distintos angulos.

Este prototipo usa herramientas gratuitas y de codigo abierto, por lo que resulta adecuado para contextos educativos de bajo costo.
